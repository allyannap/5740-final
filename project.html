# recoding primary diagnosis (encoded) into 8 meaningful groups
diag_category <- function(code) {
  if (is.na(code)) {
    return("Other")
  }
  code <- as.character(code)
  if (startsWith(code, "V") || startsWith(code, "E")) {
    return("Other")
  }
  num <- suppressWarnings(as.numeric(code))
  if (is.na(num)) {
    return("Other")
  } else if ((num>=390 & num<=459) | num==785) {
    return("Circulatory")
  } else if ((num>=460 & num<=519) | num==786) {
    return("Respiratory")
  } else if ((num>=520 & num<=579) | num==787) {
    return("Digestive")
  } else if (floor(num)==250) {
    return("Diabetes")
  } else if (num>=800 & num<=999) {
    return("Injury")
  } else if (num>=710 & num<=739) {
    return("Musculoskeletal")
  } else if ((num>=580 & num<=629) | num==788) {
    return("Genitourinary")
  } else if (num>=140 & num<=239) {
    return("Neoplasms")
  } else {
    return("Other")
  }
}

The main logistic regression model was extended by adding HbA1c and interaction terms. These different models were then compared to each other. A likelihood ratio test showed that HbA1c did not significantly improve model fit (p=0.32). However, the interaction model significantly reduced deviance compared to the HbA1c model (p<0.001), suggesting improved fit.

Due to class imbalance (only ~8.6% of patients were readmitted), using a default classification threshold of 0.5 resulted in a model that failed to identify ny readmissions. To address this, we selected a lower threshold (~0.072) based on the ROC curve to achieve approximately 70% sensitivity.

At this threshold, the cost of reduced specificity (~47%) and precision (~11%), reflected an increase in false positives. However, in a healthcare context, false negatives (missed readmissions) are more costly than false positives, making may deem this tradeoff appropriate. The model achieved an AUC of 0.623, indicating a modest predictive performance.
